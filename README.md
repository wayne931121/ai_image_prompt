# ai_image_prompt

## use on grok, gemini or vheer

當你想要仿製圖片，你可以直接請ai仿製或問她:
```
這張圖片詳細描述 包含臉部 光影 顏色 服裝 姿勢
```

然後把提示詞看要不要修改再丟給她生圖
